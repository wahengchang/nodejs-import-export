# Nodejs-import-export

# Setup

### 1- Install
```
$  yarn add babel-cli babel-preset-env
or 
$ npm install --save babel-cli babel-preset-env
```

### 2- Setup `.babelrc`
```js
{
    "presets": ["env"]
}
```

### 3- Run
```
$ ./node_modules/.bin/babel-node index.js
```


# Run
```
$ npm start
```

# Test
```
$ npm test
```
